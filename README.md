# News
News Site Using MVC 5, Entity Framework, BootStrap, SQL


News Site Using MVC 5, Entity Framework, BootStrap, SQL.

This site is build using the following: 
ASP.Net 
Identity Framwork 
.Net Framework 4.6.1 
Entity Framework 
Bootstrap 
JQuery 
AJAX

Developed using VS 2017 & MS SQL 2017

User Can Create News and can view the News.
